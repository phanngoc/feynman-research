# Feynman Research Reports

AI-powered academic research reports generated by [Feynman](https://github.com/getcompanion-ai/feynman) — a multi-agent research system running on Claude.

## What is Feynman?

Feynman is an open-source multi-agent shell for academic research:

- **Researcher agents** — search papers, repos, and web in parallel
- **Writer agent** — synthesizes findings into structured drafts
- **Verifier agent** — checks every citation URL
- **Reviewer agent** — simulates peer review
